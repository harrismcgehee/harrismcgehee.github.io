future + purrr

2017/10/03

This is just a trial/demo of an amazing blog post:

http://www.jottr.org/2017/06/the-many-faced-future.html

library("purrr")
library("future")
plan(multiprocess)

#serial
system.time(
  1:4 %>%
    map(~Sys.sleep(2))
)
##    user  system elapsed 
##   0.001   0.001   8.011
#parallel
system.time(
  1:4 %>%
    map(~ future(Sys.sleep(2))) %>%
    values
)
##    user  system elapsed 
##   0.043   0.021   2.056
